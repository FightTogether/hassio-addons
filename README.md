### 安装方法

可以用以下方式对仓库进行添加：

在家庭助手中进行设置，配置--加载项--加载项商店--仓库--“管理加载项Repo“页面中添加仓库的URL：

https://gitea.why360.cn/wenghy/hassio-addons

添加后需要刷新网页，就可以找到此加载项，点击想要安装的加载项，进行安装即可。
